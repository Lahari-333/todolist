# todolist

APP PREVIEW:  https://lahari-333.github.io/todolist/
This project is a Multi-Tenant Task Management System designed to allow multiple organizations to manage tasks securely within a single platform. Each user belongs to a specific organization, and strict data isolation is maintained using an organization ID. The system implements role-based access control, where admins can manage all tasks within their organization, while members can manage only their own tasks. It supports full CRUD operations, task filtering, and priority management. An audit log tracks all task-related actions for transparency. The application demonstrates key concepts of multi-tenancy, RBAC, and scalable SaaS architecture in a user-friendly interface.
